# PGYS — Produit

## Objectif produit

PGYS doit proposer une offre claire autour de trois piliers :

1. Cloud privé et partage de fichiers.
2. Applications métier personnalisées.
3. Hébergement, maintenance et sauvegarde.

L’objectif n’est pas de vendre de la technologie brute, mais des solutions numériques prêtes à l’usage.

## Produits principaux

### PGYS Cloud

Service de stockage, synchronisation et partage de fichiers.

Fonctions attendues :

- stockage de fichiers ;
- synchronisation ordinateur et mobile ;
- partage sécurisé ;
- gestion des utilisateurs ;
- quotas ;
- sauvegardes ;
- accès via `cloud.pgys.fr`.

PGYS Cloud repose techniquement sur Nextcloud, mais la communication commerciale doit parler de PGYS Cloud, pas de Nextcloud.

### PGYS Apps

Applications métier personnalisées développées pour les clients.

Exemples :

- gestion d’interventions ;
- suivi de chantiers ;
- CRM léger ;
- gestion documentaire ;
- extranet client ;
- tableaux de bord ;
- outils internes.

### PGYS Hosting

Hébergement de sites, applications et services.

Exemples :

- sites vitrines ;
- WordPress ;
- applications web ;
- bases de données ;
- reverse proxy ;
- supervision.

### PGYS Backup

Service futur de sauvegarde externalisée.

Objectif :

- sauvegarder les fichiers critiques des clients ;
- proposer une restauration simple ;
- créer une offre récurrente complémentaire.

## MVP commercial

Le MVP doit permettre de vendre rapidement sans automatiser tout le cycle.

Inclus :

- site vitrine PGYS ;
- page PGYS Cloud ;
- formulaire de contact ;
- création manuelle des comptes ;
- hébergement Nextcloud fonctionnel ;
- sauvegarde ;
- monitoring ;
- trois offres tarifaires simples.

Non inclus au MVP :

- paiement automatique Stripe ;
- provisioning automatique ;
- dashboard client complet ;
- SSO ;
- marketplace ;
- application mobile PGYS spécifique.

## Offre applications métier

Les applications métier sont vendues :

- au forfait ;
- ou avec abonnement maintenance ;
- ou avec hébergement inclus.

Modèle recommandé :

- frais de développement initial ;
- abonnement mensuel pour hébergement, maintenance et support.

## Proposition de valeur

PGYS aide les petites entreprises à disposer d’outils numériques fiables, simples et adaptés à leur métier, sans gérer la complexité technique.
