# PGYS — Vision

## 1. Résumé

PGYS est une marque française de solutions numériques pour les indépendants, TPE, PME et associations.

PGYS propose une approche simple : fournir aux petites structures des outils fiables, hébergés, maintenus et adaptés à leur activité, sans leur imposer la complexité d’un grand système informatique.

PGYS ne se limite pas au cloud. Le cloud est une première brique d’un écosystème plus large comprenant :

* PGYS Cloud ;
* applications métier personnalisées ;
* hébergement ;
* sauvegardes ;
* maintenance ;
* accompagnement technique.

L’objectif est de devenir un partenaire numérique de proximité pour les petites entreprises.

---

## 2. Origine du nom

PGYS vient de l’idée initiale **PROGYS**, inspirée du mot **progiciel**.

Le nom PGYS conserve cette origine tout en étant plus court, plus distinctif et plus adapté à une marque numérique moderne.

PGYS évoque :

* les progiciels ;
* la gestion ;
* les systèmes ;
* les solutions professionnelles ;
* la simplicité d’un nom court.

Le domaine `pgys.fr` est court, mémorisable et adapté à une marque principale.

---

## 3. Vision à long terme

À long terme, PGYS doit devenir une plateforme de services numériques pour petites entreprises.

Une entreprise cliente doit pouvoir utiliser PGYS pour gérer :

* ses fichiers ;
* ses sauvegardes ;
* ses applications métier ;
* son hébergement ;
* sa maintenance ;
* ses outils numériques internes.

La vision n’est pas de concurrencer Microsoft, Google ou OVH sur leur terrain.

La vision est de proposer une alternative de proximité, plus simple, plus humaine et mieux adaptée aux petites structures.

---

## 4. Mission

La mission de PGYS est de permettre aux petites entreprises de disposer d’outils numériques professionnels sans avoir besoin d’un service informatique interne.

PGYS doit rendre accessibles :

* le cloud privé ;
* les applications métier ;
* l’hébergement ;
* la maintenance ;
* la sauvegarde ;
* le support technique.

Le client doit pouvoir se concentrer sur son métier, pas sur son infrastructure.

---

## 5. Positionnement

PGYS se positionne comme un partenaire numérique pour les entreprises.

Le message principal :

> Des solutions numériques simples, fiables et adaptées aux petites entreprises.

PGYS n’est pas seulement un fournisseur technique. PGYS accompagne, conseille, installe, maintient et fait évoluer les outils numériques de ses clients.

---

## 6. Produits actuels

### PGYS Cloud

Service de stockage, synchronisation et partage de fichiers, basé sur une infrastructure maîtrisée.

Objectifs :

* proposer un cloud privé ;
* héberger les données en France ;
* simplifier le partage de fichiers ;
* offrir un support humain ;
* créer une source de revenu récurrent.

### Applications métier personnalisées

Développement d’outils spécifiques pour répondre aux besoins concrets d’un client.

Exemples :

* gestion d’interventions ;
* gestion de clients ;
* suivi de chantiers ;
* gestion de documents ;
* tableaux de bord métier ;
* outils internes.

### Hébergement

Hébergement de sites, services web, applications métier et outils clients.

### Maintenance

Support, mises à jour, supervision, sécurité et accompagnement.

---

## 7. Produits futurs

### PGYS Backup

Service de sauvegarde externalisée pour postes, serveurs et documents critiques.

### PGYS Mail

Messagerie professionnelle simple, maintenue et intégrée à l’écosystème PGYS.

### PGYS Apps

Portail centralisant les applications métier des clients.

### PGYS Identity

Gestion des accès, utilisateurs, permissions et authentification.

### PGYS Workspace

Espace numérique complet regroupant fichiers, applications, comptes, support et services.

## 8. Clients cibles

PGYS vise principalement :

* indépendants ;
* artisans ;
* TPE ;
* PME ;
* associations ;
* professions libérales ;
* structures sans service informatique interne.

Ces clients recherchent généralement :

* une solution simple ;
* un interlocuteur humain ;
* un prix clair ;
* un accompagnement ;
* une infrastructure fiable ;
* une solution adaptée à leur métier.

---

## 9. Valeurs

### Simplicité

Une solution PGYS doit être compréhensible rapidement.

Si une fonctionnalité demande trop d’explications, elle doit être simplifiée.

### Fiabilité

Un service PGYS doit être stable, sauvegardé, supervisé et maintenable.

### Transparence

Les prix, limites, responsabilités et conditions doivent être clairs.

### Souveraineté

Les données doivent rester sous contrôle, avec une préférence pour l’hébergement en France.

### Proximité

PGYS doit conserver un support humain et accessible.

### Utilité

Chaque fonctionnalité doit répondre à un besoin réel d’un client.

---

## 10. Philosophie produit

PGYS doit privilégier les produits utiles, sobres et durables.

Le but n’est pas d’ajouter un maximum de fonctionnalités, mais de résoudre correctement des problèmes concrets.

Une fonctionnalité est pertinente si elle :

* répond à un besoin client identifié ;
* simplifie le quotidien ;
* améliore la fiabilité ;
* réduit le temps perdu ;
* augmente la valeur perçue du service.

---

## 11. Philosophie technique

La technologie est un moyen, pas une finalité.

PGYS privilégie :

* Docker pour la portabilité ;
* Git pour la traçabilité ;
* TypeScript pour la robustesse ;
* PostgreSQL pour les données applicatives ;
* Nextcloud comme moteur cloud ;
* Next.js pour les interfaces web ;
* NestJS pour les API structurées ;
* une documentation claire ;
* des sauvegardes testées ;
* une architecture simple avant toute complexité.

PGYS évite :

* les architectures prématurément complexes ;
* Kubernetes au démarrage ;
* les microservices inutiles ;
* les dépendances non maîtrisées ;
* les secrets dans le code ;
* le développement directement en production.

---

## 12. Promesse

PGYS promet aux petites entreprises des solutions numériques simples, fiables et accompagnées.

La promesse commerciale :

> PGYS vous aide à gérer vos outils numériques sans complexité, avec un interlocuteur humain et des solutions adaptées à votre activité.

---

## 13. Principe stratégique

PGYS doit construire une relation client durable.

Le cloud permet d’entrer chez le client avec un abonnement simple.

Les applications métier permettent ensuite d’apporter une forte valeur ajoutée.

L’hébergement et la maintenance permettent de fidéliser.

La stratégie est donc :

1. attirer avec une offre simple ;
2. rassurer avec un service fiable ;
3. développer la relation avec des applications personnalisées ;
4. fidéliser avec l’hébergement, la maintenance et le support.

---

## 14. Vision à 5 ans

Dans 5 ans, PGYS doit pouvoir proposer une plateforme complète où un client peut :

* gérer ses fichiers ;
* accéder à ses applications métier ;
* consulter ses sauvegardes ;
* suivre ses services hébergés ;
* contacter le support ;
* consulter ses factures ;
* gérer ses utilisateurs.

PGYS doit devenir le point d’entrée numérique principal de ses clients professionnels.
