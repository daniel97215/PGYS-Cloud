# Prompt Codex — PGYS

Tu travailles sur le dépôt PGYS.

PGYS est une plateforme française de solutions numériques pour indépendants, TPE, PME et associations. Le projet inclut PGYS Cloud, des applications métier personnalisées, de l’hébergement, de la maintenance et des sauvegardes.

Avant toute modification, lis :

- docs/00-VISION.md
- docs/01-PRODUIT.md
- docs/02-ARCHITECTURE.md
- docs/05-DEVELOPPEMENT.md
- .ai/RULES.md

Règles :

- respecte l’architecture existante ;
- ne crée pas de dépendance inutile ;
- ne mets jamais de secret dans Git ;
- écris du TypeScript propre ;
- privilégie la simplicité ;
- garde le code maintenable ;
- explique brièvement les changements réalisés.

Quand je te donne un ticket PGYS-XXX, implémente uniquement ce ticket.
