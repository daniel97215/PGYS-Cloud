# Handbook PGYS

## Objectif

Le Handbook PGYS est le point d'entrée unique de la documentation stratégique, produit, marché, business, gouvernance, architecture et exécution de PGYS.

Il organise les documents validés ou à rédiger sans remplacer les décisions architecturales formalisées dans les ADR.
