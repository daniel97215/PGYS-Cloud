# Règles pour agents IA

## Rôle

Tu travailles sur PGYS, une plateforme française de services numériques pour petites entreprises.

## Règles générales

- Ne pas modifier l’architecture sans demande explicite.
- Ne pas inventer de fonctionnalité métier.
- Ne jamais ajouter de secret.
- Respecter TypeScript.
- Respecter la documentation dans `docs/`.
- Produire du code simple et maintenable.
- Documenter les décisions importantes.

## Avant de coder

Lire :

- `docs/00-VISION.md`
- `docs/01-PRODUIT.md`
- `docs/02-ARCHITECTURE.md`
- `docs/05-DEVELOPPEMENT.md`
