# Exécution

## Rôle de la section

La section Exécution regroupe les documents opérationnels utilisés pour suivre le travail courant.

Ces documents ne remplacent pas les documents stratégiques du Handbook. Ils servent à traduire les orientations validées en éléments suivables.

## Dossiers

- [backlog/](backlog/) : éléments de backlog et demandes à traiter.
- [tickets/](tickets/) : tickets de travail détaillés.
- [plans-de-livraison/](plans-de-livraison/) : plans, jalons opérationnels et notes de livraison.
