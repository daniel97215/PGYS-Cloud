# PGYS — Branding

## Nom

La marque principale est PGYS.

Les produits sont nommés de manière simple :

- PGYS Cloud
- PGYS Apps
- PGYS Hosting
- PGYS Backup
- PGYS Mail
- PGYS Workspace

## Positionnement de marque

PGYS doit inspirer :

- confiance ;
- simplicité ;
- proximité ;
- sérieux ;
- maîtrise technique.

PGYS ne doit pas paraître comme un simple projet personnel ou un hébergeur amateur.

## Message principal

> Des solutions numériques simples pour les entreprises.

Variantes :

> Votre partenaire numérique de proximité.

> Cloud, applications et hébergement pour petites entreprises.

> Des outils professionnels, simples et maintenus.

## Ton

Le ton doit être :

- clair ;
- professionnel ;
- rassurant ;
- direct ;
- humain.

Éviter :

- jargon inutile ;
- promesses excessives ;
- vocabulaire trop startup ;
- formulations trop techniques.

## Couleurs

Palette initiale :

- Bleu principal : `#2563EB`
- Bleu foncé : `#1E3A8A`
- Texte principal : `#0F172A`
- Gris texte : `#475569`
- Fond clair : `#F8FAFC`
- Blanc : `#FFFFFF`
- Vert succès : `#10B981`
- Rouge erreur : `#EF4444`

## Typographie

Police recommandée :

- Inter

Alternatives :

- Manrope
- Plus Jakarta Sans

## Logo

Le logo initial peut être typographique :

```text
PGYS
```

ou :

```text
PGYS Cloud
```

Un symbole pourra être ajouté plus tard :

- hexagone ;
- nuage ;
- cube ;
- monogramme P.

## Hiérarchie de marque

`pgys.fr` est le site principal.

Sous-domaines :

- `cloud.pgys.fr`
- `admin.pgys.fr`
- `api.pgys.fr`
- `status.pgys.fr`
- `apps.pgys.fr`

## Principes UI

- beaucoup d’espace ;
- interfaces sobres ;
- boutons lisibles ;
- cartes simples ;
- contraste élevé ;
- responsive mobile ;
- accessibilité.

## Éléments à éviter

- design trop chargé ;
- effets visuels excessifs ;
- animations inutiles ;
- vocabulaire trop technique ;
- mise en avant de Nextcloud comme marque principale.
